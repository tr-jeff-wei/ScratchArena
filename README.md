# ScratchArena
歡迎來到 ScratchArena —— 專為新世代打造的積木程式電競平台！我們將傳統的程式測驗轉化為熱血的遊戲競技場。在這裡，學生不只是在解題，而是透過編寫 Scratch 邏輯來控制角色、破關解謎、進行 AI 代理人即時對戰。結合天梯排行榜與每日擂台賽，ScratchArena 讓程式學習告別枯燥，用積木引爆競爭力，寫出你的最強王牌！

Welcome to ScratchArena — the ultimate block-coding esports platform! We transform traditional coding tests into a thrilling gaming arena. Here, students don't just solve problems; they write Scratch logic to control characters, crack puzzles, and engage in real-time AI battles. Featuring ranked leaderboards and daily challenges, ScratchArena turns coding into a competitive sport, sparking logic and creativity block by block!
